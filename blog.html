Starting with the point that everything is possible this project will dive into the sea that is object detection. A process of identifying, localizing, and classification of an object in a scene that is object detection which is a branch of computer vision
The spread of powerful computers, the availability of high quality, cheap video cameras and the highly need for automated video analysis has created a focused interest in object tracking algorithms. Video analysis has three main steps as: detecting a specific moving objects, tracking an objects between frames and recognize object behavior by analysis their tracks. So object tracking is relevant to the following tasks:
  • Recognition based on motion, as identify someone based on gait and automatic object detection, etc.
  • Automated surveillance as Detection of suspect activities or unforeseen from a scene.
  • Video indexing as retrieving automatic annotation of the videos in databases of multimedia.
  • Human interaction with the computer as gesture recognition, tracking eye gaze to be stored in computers etc.
  • Traffic monitoring as gathering traffic statistics in real time to direct traffic flow.
  • Vehicle navigation as planning path based video and obstacle avoidance capabilities.
